# DurableTask UI

React SPA for managing and monitoring Durable Task orchestrations.

## Development

```bash
pnpm install
pnpm start
```

## Build

```bash
pnpm build
```

## E2E Tests

Tests use Playwright with API mocking (no backend needed).

```bash
# First time: install browser
pnpm exec playwright install chromium

# Run tests
pnpm test
```
